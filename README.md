# SEO-frontend-ts-nextjs

A Next.js + TypeScript project with **SEO best practices** including:

- Metadata management (title, description, Open Graph, Twitter Cards)
- Sitemap generation
- Robots.txt configuration
- Google Search Console integration
- Caching strategies for better performance
